== Description ==

This widget gives you ability to show top online radio stations of Volnorez.com, located at http://www.volnorez.com.

== Installation ==

1. Upload and extract VolnorezOnline.zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add plugin to the sidebar

== Frequently Asked Questions ==

= Do I need to have an account on Volnorez.com? =

No, the plugin does not require any login.
